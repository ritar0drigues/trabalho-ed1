# 📋 Sistema de Gerenciamento de Tarefas

## 📚 Requisitos do Projeto

### Estruturas de Dados Fundamentais

#### 1. 📝 Lista Encadeada
- Armazenamento principal de tarefas
- Cada nó contém:
  - Nome da tarefa
  - Prioridade
  - Deadline

#### 2. 🔄 Lista Encadeada Circular
- Histórico de tarefas concluídas
- Capacidade limitada (últimas 10 tarefas)
- Mantém registro circular das conclusões

#### 3. ⚖️ Lista Duplamente Encadeada
- Organização por prioridade
- Reordenação dinâmica de tarefas
- Critérios flexíveis de ordenação

### Estruturas Auxiliares

#### 4. ↩️ Pilha
- Sistema de desfazer ações
- Funcionalidades:
  - Reverter remoção de tarefas
  - Desfazer alterações de prioridade

#### 5. 📊 Fila
- Gerenciamento de tarefas pendentes
- Ordenação por:
  - Ordem de chegada
  - Critérios personalizados

### Algoritmos

#### 6. 🔍 Recursividade
- Busca recursiva de tarefas
- Localização por:
  - Palavras-chave
  - Nome da tarefa
  - Descrição

#### 7. 📈 Algoritmos de Ordenação
- Múltiplos critérios de ordenação:
  - Data de criação
  - Prioridade
- Implementação de diversos métodos de ordenação

#### 8. 🔎 Algoritmos de Busca
- Busca binária
- Localização rápida após ordenação
- Otimização de busca
